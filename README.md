# To-Do List App

## Description
A responsive To-Do List application built using HTML, CSS, Bootstrap, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Local Storage
- Responsive Design

## Technologies
- HTML
- CSS
- Bootstrap 5
- JavaScript

## How to Run
Open index.html in your browser.
